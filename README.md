# Vikneshwaran Portfolio

A modern and interactive developer portfolio built with Next.js, React.js, and Tailwind CSS. This portfolio showcases my projects, technical skills, LeetCode progress, experience, and AI-powered contact assistant.

---

## 🚀 Features

- Modern responsive UI/UX
- Interactive animations and transitions
- AI-powered email assistant
- LeetCode progress integration
- Project showcase with live demos
- Skills & technologies section
- Experience and education timeline
- Resume download support
- Dark aesthetic with premium typography
